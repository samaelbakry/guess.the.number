# Guess The Number

A small Expo React Native game where the app guesses the number you select and you guide it by choosing whether the secret number is higher or lower.

## What it does
- Lets the player enter a number between 1 and 99
- The app makes a starting guess
- The player taps **Higher** or **Lower** until the app finds the number
- Shows the number of rounds and the chosen number at game over

## Tech stack
- Expo Router
- React Native
- TypeScript
- Tailwind-style styling with `twrnc`
- Expo vector icons and native safe area support

## Notes
- The game is built for a simple single-player guessing experience
- Restart from the game over screen to play again
